UCMR5 Water Quality API

Access EPA UCMR5 water quality data by ZIP code, PWSID, or location.

API Endpoints

Standard API

GET /api/water-quality?zipcode=12345
GET /api/water-quality?pwsid=NY0000123
GET /api/water-quality?pws_name=Springfield
GET /api/water-quality?state=NY

Zapier-Optimized API

Use this endpoint for better Zapier integration with flattened fields:

GET /api/water-quality-zapier?zipcode=12345
GET /api/water-quality-zapier?pwsid=CA0110008

Try It

  • Search by ZIP: 10001 (NYC)
  • Search by PWSID: NY7003493
  • Search by Name: Albany in NY
